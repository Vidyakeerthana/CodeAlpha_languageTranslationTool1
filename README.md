# CodeAlpha_languageTranslationTool1
This is a Python-based Language Translation Tool built as a task of internship project at CodeAlpha. The tool allows users to translate text between multiple languages and includes a text-to-speech feature. It wil run in Google Colab, making it accessible without any local Python setup.

Features
- Translate text between multiple languages using Google Translate API 
- Text-to-Speech (TTS) feature using gTTS 
- User-friendly GUI interface using ipywidgets in Colab
- Works completely online, no local Python installation needed 
- Supports languages: English, Hindi, Tamil, French, German, Japanese 
- Easy to use
